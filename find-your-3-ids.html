Find Your 3 IDs

Use these 3 values to configure the Voice AI widgets and in-page orbs: Widget ID, Location ID, and Agent ID.

Back to Widget Generator Open In-Page Orb Demo

1. Find Your Widget ID

How to find your Widget ID

  1. Log in to your LeadConnector account.
  2. Navigate to Settings ‚Üí Voice AI.
  3. Select your Voice AI widget.
  4. Copy the Widget ID from the configuration panel.

2. Find Your Location ID

How to find your Location ID

  1. Log in to your LeadConnector account.
  2. Navigate to Settings ‚Üí Business Profile.
  3. Your Location ID appears in the URL or under API Access.
  4. Copy the Location ID value.
Location ID example (`locationid.gif`)
Click the image to enlarge.

3. Find Your Agent ID

How to find your Agent ID

  1. Log in to your LeadConnector account.
  2. Navigate to Settings ‚Üí Voice AI.
  3. Select your Voice AI agent/configuration.
  4. Copy the Configuration ID (`voiceAiConfigId`).
