Find Your 3 IDs

Use these 3 values to configure the Voice AI widgets and in-page orbs for your own website: Widget ID, Location ID, and Agent ID.

1. Find Your Widget ID

Follow these steps within LeadConnector

  1. Log in to your relevant sub-account.
  2. Navigate to Settings → Voice AI.
  3. Select the Voice AI widget you created.
  4. Copy the Widget ID from the right-hand configuration panel.

2. Find Your Location ID

Follow these steps within LeadConnector

  1. Ensure you are in the correct sub-account.
  2. Navigate to Settings → Business Profile.
  3. Your Location ID appears in your browser's URL bar, or under API Access.
  4. Copy this value.

3. Find Your Agent ID

Follow these steps within LeadConnector

  1. Navigate to Settings → Voice AI.
  2. Select the specific Voice AI agent configuration you want to use.
  3. Look for the Configuration ID (also known as `voiceAiConfigId`).
  4. Copy this value.
